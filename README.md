# NFTMania

***

## Features

- Using Express and NodeJS as a backend.
- Using VueJS as a frontend.
- Using MongodDB as database.

***

## Prerequisites

- **NodeJS >= v14.x.x**
- **NPM >= v6.x.x**
- **MongoDB = v5.x.x**
- **Express = v4.x.x**
- **VueJS = v3**

## Installation

1. Clone repo of the project
    ```js
    git clone https://github.com/dCodeX21/nftmania.git
    ```
2. Access API directory
    ```js
    cd api
    ```
3. Install dependencies
    ```js
    npm install
    ```
4. Access App directory
    ```js
    cd ../app
    ```
5. Install dependencies
    ```js
    npm install
    ```
    
***

## Usage

\*In the API directory
```js
npm start
```

\* In the App directory
```js
npm run serve
```
    
Verify the deployment by navigating to your server address in your preferred browser.

\*For API
```js
localhost:3000
```
\*For App
```js
localhost:8080
```
